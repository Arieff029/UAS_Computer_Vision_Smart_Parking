# 🚗 Smart Parking System menggunakan YOLOv8 dan OpenCV

Proyek ini merupakan proyek **UAS Computer Vision** untuk mendeteksi ketersediaan slot parkir secara real-time berbasis kamera/CCTV statis.

---

### 🌟 1. Fungsi / Fitur
* **Real-time Monitoring:** Memantau ketersediaan slot parkir secara langsung dari input video.
* **Smart Indicator:** Kotak pembatas parkir otomatis berubah warna secara interaktif (Hijau = Kosong, Merah = Terisi).
* **Live Counter:** Menampilkan jumlah total slot kosong yang tersedia di pojok layar secara real-time.
